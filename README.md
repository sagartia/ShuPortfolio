# SHU Portfolio Website

這是一個以 Vue 3 製作的個人履歷與作品集網站，主要用來展示我的技術棧、Side Projects、開發經歷與聯絡資訊。網站風格採用深色系介面，搭配翡翠綠與藍色作為科技感亮點，目標是讓技術主管與 HR 能快速理解我的工程能力與專案方向。

## 網站內容

- Hero Section：呈現個人專業定位與主要行動按鈕
- About & Skills：介紹學習背景與技術棧分類
- Side Projects：展示電商平台、自動化訂單流程與作品集網站
- Experience：以時間軸整理開發經歷與里程碑
- Contact：提供 GitHub、LinkedIn 與 Email 聯絡入口

## 技術棧

- Vue 3
- Composition API `<script setup>`
- Vite
- Tailwind CSS
- Lucide Vue Next
- RWD 響應式設計
- Dark Mode Base 深色視覺設計

## 專案特色

- 使用資料陣列管理技術棧、專案列表與經歷內容，方便後續修改
- 使用 `v-for` 渲染重複區塊，降低 HTML 重複度
- 單頁式 Portfolio 結構，瀏覽動線清楚
- 導覽列固定於頂部，方便快速切換區塊
- 支援手機、平板與桌面版版面
- 使用 Lucide 圖標提升介面識別度
